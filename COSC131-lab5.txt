DD
constants for menu choices:  PKG_A, PKG_B, PKG_C, QUIT
constants for base prices:  PKG_A_BASE, PKG_B_BASE, PKG_C_BASE
constants for hours of access:  PKG_A_HRS, PKG_B_HRS, PKG_C_HRS
constants for additional hours rates:  PKG_A_ADD, PKG_B_ADD, PKG_C_ADD
constants for hours used:  MIN_HRS, MAX_HRS
choice - menu choice, integer, input value
hours - hours used, real, input value
baseCost - cost of the package, real
addHrs - additional hours used, real
addCharges - additional charges, real
total - total amount due, real
Pseudocode
Display menu and input choice
Select on choice
  case PKG_A:
     Prompt for and input hours 
     If hours value is valid
         Set baseCost to PKG_A_BASE
         Set addHrs to hours minus PKG_A_HRS
         If addHrs is negative
             Set addHrs to zero
         Set addCharges to addHrs times PKG_A_ADD,
         Set total to baseCost plus addCharges
         Display total amount due
     Else
          Display message about invalid hours
   case PKG_B:
        // your task to complete
   case PKG_C:
        // your task to complete
   case QUIT:
        // your task to complete
   default:
        // your task to complete
